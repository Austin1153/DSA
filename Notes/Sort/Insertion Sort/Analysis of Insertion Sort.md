```cpp
template <class T>
void Insert(const T &e, T *a, int i) { //i = j-1 
	a[0] = e;
	while (e < a[i]) { //i = 1 ~ n-1(i -> i+1 comparisons)
		a[i+1] = a[i];
		i--;
	}
	a[i+1] = e;
}
```
$$O( \sum_{i=1}^{n-1}(i+1)) = O(n^2)$$
